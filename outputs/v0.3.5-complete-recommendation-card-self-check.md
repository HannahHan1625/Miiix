# v0.3.5 Complete Recommendation Card Self Check

Date: 2026-07-06

## Scope

- Fix the recommendation modal tutorial card clipping issue.
- Preserve the page-level centered popup.
- Keep recipe actions visible after flip.

## Implementation

- Bumped app version to `v0.3.5`.
- Added `resultCardBody` wrappers to the front and back faces of the recommendation card.
- Changed the modal card layout to `content body + fixed action area`.
- Allowed only the content body to scroll when recipe content grows.
- Increased the card height to a mobile-safe range while keeping it capped by viewport height.

## Product Judgment

This is not a decorative fix. The recommendation card is the point where users decide whether to cook, save, or plan a dish. If the action area is clipped, the core loop breaks. The card now treats recipe instructions as variable content and actions as fixed commitments.
